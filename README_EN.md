## What is this?

Love Director is a **relationship pattern observer and director system**.

Not a dating coach. Not a fraud detector (that's [emotional-fraud-detector](https://github.com/awaqufexituf65-cmyk/emotional-fraud-detector)). Not a therapist.

Love Director is a **director** — it takes your real chat data and national relationship statistics, then stages multiple possible futures for your love life. It doesn't tell you what to do. It lets you **see** where different choices might lead.

---

## Core Features

- 🧬 **Three-Layer Profile**: Self-report questionnaire + WeFlow WeChat behavior extraction + gap analysis
- 💭 **Seven-Dimension Philosophy Probe**: Time beliefs / Relationship purpose / Value hierarchy / Giving patterns / Conflict beliefs / Social media influence / Security attachment
- 📊 **Behavioral Observation**: Combined with wechat_analysis sentiment curves + cross-relationship pattern comparison
- 🌳 **Probabilistic Decision Tree**: Multiple love-life trajectories grown from your data, each with probability estimates and data evidence
- 🎬 **Director's Perspective**: Six ending types (Inertia / Growth / Surprise / Warning / Cycle / Reconstruction) told as narrative stories
