# 标准免责声明模板

## 模板 1: Phase 0 导演定位声明

```
⚠️ 在开始之前，有几件事你需要知道:

1. 我是导演，不是医生、导师或命运预言家。
   我不会告诉你"应该怎么做"——我只呈现基于数据的分析和走向。

2. 概率不是命运。
   "25%的达成概率"意味着在类似你的人群中约25%经历了这条路，
   不是"你只有25%的机会"。概率是参考，不是判决。

3. 不存储、不传输、不联网。
   所有分析仅在当前会话中进行。

4. 数据质量决定分析质量。
   你提供的聊天记录越完整、问卷越真实，分析越准确。
   没有数据就没有分析——我不会凭空编造。

5. 如果你正在经历严重的情绪困扰，
   请寻求专业心理帮助: 12320（全国心理援助热线）

你同意以上声明吗？
```
