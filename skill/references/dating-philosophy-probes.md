# 七维恋爱哲学探针详述

## 设计原则

1. **不直接问抽象问题** — "你觉得金钱重要还是感情重要"得到的永远是标准答案。用场景回溯。
2. **锚定具体经历** — "你上次拒绝一个人是为什么？"比"你的择偶标准是什么？"真实得多。
3. **自述 + WeFlow 验证** — 你说的 vs 你做的，差距本身就是发现。

---

## 维度 1: 时间信念

### 探测问题

| 问题 | 揭示什么 |
|------|---------|
| 从认识到确认关系你一般需要多久？最短那次多久？ | 确认速度基线 |
| 你觉得认识多久见家长合适？ | 进程预期 |
| 你有没有"考察期"这个概念？考察什么？多久？ | 信任建立方式 |
| 如果有人认识第2天就表白，你怎么想？ | 对"快"的容忍度 |
| 聊了半年对方还没明确表态，你怎么想？ | 对"慢"的焦虑度 |

### 检测的畸变
- ⚡ 闪电型 (<3天确认) — 缺乏筛选机制
- 🐌 无限考察型 (>6个月在"考察") — 用考察回避承诺
- 📏 严格时间表型 — 把关系当项目管理

### 健康对照
Gottman: 关系发展没有统一时间表。Altman & Taylor 社会渗透理论: 亲密感通过渐进披露建立，这个过程无法被加速。

## 维度 2: 关系目的

### 探测问题

| 问题 | 揭示什么 |
|------|---------|
| 你最近开始一段关系，是因为喜欢这个人，还是因为想谈恋爱？ | 动机类型 |
| 如果未来3年不结婚，你的恋爱方式会变吗？ | 婚姻压力驱动程度 |
| 你上一段恋爱"成功"或"失败"的标准是什么？ | 目标定义 |
| 你有没有单纯为了"不想一个人"而在一起过？ | 孤独驱动 |
| 如果全社会明天开始都不结婚，你还会恋爱吗？ | 恋爱本身的价值 |

### 目的分类

| 类型 | 特征 | 风险 |
|------|------|------|
| 🎯 婚姻导向 | 每段都在评估"能不能结婚" | 过度筛选→没人符合→焦虑加剧 |
| 🌊 体验导向 | 享受过程，不急于定义结果 | 可能回避承诺 |
| 🛟 逃避驱动 | 用恋爱逃离孤独/压力/上一段 | 关系基础脆弱 |
| 🔄 替代驱动 | 工作不如意→恋爱；生活无聊→恋爱 | 关系变成填补空缺的工具 |
| 🧭 成长导向 | 互相成长，对过程享受、对未来开放 | 最健康的类型 |

## 维度 3: 价值排序

### 行为回溯法 (不用直接问)

```
请回忆你最近一次拒绝一个人(或分手)。
以下是你当时考虑的真正原因，请按重要性排序:

[ ] TA的经济条件不满足我的底线
[ ] TA的外貌/身高不符合我的要求
[ ] TA的性格让我不舒服
[ ] TA的家庭背景不符合我的预期
[ ] 我对TA没有感觉了
[ ] 我遇到了更好的人
[ ] TA对我不够好/不够在乎
[ ] 我们未来的规划不一致
[ ] TA的学历/工作不够好
[ ] 家人/朋友不认可
```

### 为什么行为回溯更准
你自称看重的东西在真实决策中可能排在第五位。你实际因什么拒绝别人，才是你的真实价值排序。

### WeFlow 验证
对比你的分手原因自述 vs 聊天记录中的分手前对话主题分布是否一致。

## 维度 4: 付出观念

### 探测问题

| 问题 | 揭示什么 |
|------|---------|
| 一段健康的关系中，男/女方应该付出更多吗？为什么？ | 性别角色刻板 |
| 你有没有觉得"TA对我好是应该的"？哪些方面？ | 期望基线 |
| 你做没做过自己不愿意但觉得"应该做"的关系行为？ | 自我牺牲模式 |
| 你有没有过"我对TA那么好TA凭什么..."的想法？ | 付出=回报的交换思维 |

### 检测的畸变
- 🐶 舔狗模式: "只要我付出足够多TA就会爱我"
- 👸 理所当然型: "男生就该买单/女生就该被宠"
- 📊 记账型: 默默计算谁付出更多
- 🫠 自我消解型: 为讨好不断压缩边界

### 健康对照
Hatfield 公平理论: 健康亲密关系需要双方投入的平衡感。Gottman: 正面/负面互动比为5:1的夫妻最稳定。

## 维度 5: 冲突信念

### 探测问题

| 问题 | 揭示什么 |
|------|---------|
| 你觉得吵架是好事还是坏事？ | 冲突基本观念 |
| 上次你和对象意见不一致时，你怎么处理的？ | 实际冲突策略 |
| 你有没有"算了不说了，说了也没用"的时候？ | 回避触发点 |
| 你觉得什么情况值得分手？ | 分手阈值 |

### 检测的畸变
- 🏃 零冲突理想: "好的感情不会吵架"
- 🔥 冲突=在乎: "吵得越凶越在乎"
- 🤐 沉默回避: 放弃沟通
- 📏 低分手阈值 vs 🧱 高分手阈值

### 健康对照
Gottman: 69%的冲突是永久性的。不一定要解决，但要管理。修复尝试是关系韧性的关键。严文华(2024): 中国青年最常用回避策略。

## 维度 6: 社交媒体渗透

### 探测问题

| 问题 | 揭示什么 |
|------|---------|
| 你觉得理想中的恋爱，多大程度上受到社交媒体影响？ | 自觉程度 |
| 你有没有因为"别人男朋友都XX"而对关系不满？ | 比较驱动的不满 |
| 你试过短视频/小红书上的恋爱技巧吗？结果如何？ | 行为被塑程度 |
| 你平时刷到什么恋爱内容最多？(考验/技巧/吃瓜/科普) | 信息环境 |

### 检测的畸变
- 📱 模板化爱情: 用博主的标准丈量自己的关系
- 🎭 恋爱技巧依赖: 每次约会前看技巧视频
- 🍿 吃瓜型恋爱观: 看了太多"锤渣男"→过度警惕
- 💸 消费主义绑架: 秋天的第一杯奶茶、十根烤肠 = 爱

### 数据支撑
北师大(2025): 71%年轻人通过短视频学恋爱技巧，但真实恋爱率暴跌43%。每周看>10小时颜值直播→相亲成功率降67%。

## 维度 7: 安全感来源

### 探测问题

| 问题 | 揭示什么 |
|------|---------|
| 在一段关系中，什么让你觉得安心？ | 安全感锚点 |
| 什么会立刻让你觉得不安？ | 不安全感触发器 |
| 你有频繁查对方手机/定位/社交账号的经历吗？ | 控制行为 |
| 你觉得自己值得被爱吗？(不需要回答我，问自己) | 底层自尊 |

### 安全感外部化 vs 内部化

| 类型 | 锚点 | 问题 |
|------|------|------|
| 🔒 外部化 | 房产证、每日报备、朋友圈官宣 | 安全感由不可控的外部因素决定 |
| 💚 内部化 | 自己价值的确认、关系韧性信任 | 健康但不等于没有正常需求 |

### [推荐] Lila MCP 依恋风格评估
- 安全型 (~56%人口): 能够信任并依赖他人
- 焦虑型 (~19%): 担心被抛弃，需要持续确认
- 回避型 (~25%): 对亲密感到不适，保持情感距离
- 混乱型 (~5-10%): 既渴望又恐惧亲密

> 若不安装 Lila MCP，用本维度的探测问题做问卷自评替代。
